Description

Englo is a tool to manage and deliver software projects. Itâ€™s based on Entando, an Open Source web application platform, written in Java (see Entando's Github Repositories).

Features

Englo system includes two modules: englo-portal and englo-entando-components. englo-portal is able to run all the services generated from englo-entando-components module, that contains englo, basecamp, trello, github, jenkins apps.

Getting Started

  • Clone entando-core and entando-components from Entando's Github Repositories.
  • Install entando-core with the following command:

mvn install -DskipTests.

  • Install entando-components with the following command:

mvn install -DskipTests.

englo-portal can be run using either Tomcat or Jetty. Enter in the englo-portal folder; if you are using Jetty, you can run with englo-portal following: mvn clean jetty: run.
